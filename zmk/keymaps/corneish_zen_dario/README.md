## Magic Key Mappings
### BASE_PRIMARY
- default: REPEAT
- timeout_ms: 1000
- mappings:
  -   → THE
  - , →  BUT
  - - → >
  - . → /
  - C → Y
  - G → Y
  - / → *
  - * → /

### BASE_ALT
- default: REPEAT
- timeout_ms: 0
- mappings:

  - , →  BUT
  - - → >
